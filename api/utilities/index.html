    Utilities

    SplashKit provides some basic data manipulation functions to help make it easy to get some basic tasks performed.SplashKit random provides a simple implementation of random.SplashKit provides miscellaneous utilities for unclassified functionality.

    Delay

    There are multiple versions of Delay:
    • Delay ( Unsigned Integer )

    Delay

    Puts the program to sleep for a specified number of milliseconds. If this is larger than 1 second, SplashKit will check to see if the user tries to quit during the delay. If the user does quit, the delay function returns without waiting.
